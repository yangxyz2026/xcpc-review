# 独立题目

非 VP 比赛的单独练习题、专题训练题。

## 目录结构

每道题一个文件夹，命名格式：平台_题号 或 专题_序号

`
problems/
├── CF_1800D/               ← Codeforces 题
│   ├── main.cpp            ← AC 代码
│   ├── WA_1.cpp            ← 错误版本（可选）
│   └── note.md             ← 题解笔记（可选）
│
├── AtCoder_abc300_F/       ← AtCoder 题
│   └── main.cpp
│
└── 贪心_01_区间覆盖/       ← 专题训练
    └── main.cpp
`

## 文件说明

| 文件 | 说明 |
|------|------|
| main.cpp | 最终 AC 代码 |
| WA_*.cpp | 错误版本（用于回顾） |
| 
ote.md | 解题思路、踩坑记录 |
